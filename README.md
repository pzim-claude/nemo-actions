## âš¡ Quick Install

Download the `.zip` of any action and extract it to `~/.local/share/nemo/actions/`:

```bash
unzip action@pzim-devdata.zip -d ~/.local/share/nemo/actions && nemo -q
```

Or use the **one-line install** command from each action's page.
